# 8gent Dev

Developer documentation site for the 8gent ecosystem.

**Live:** [8gent.dev](https://8gent.dev)

## Stack

- **Framework:** Next.js 15 + Fumadocs
- **Deployment:** Vercel
- **Content:** MDX docs in `content/`

## What's Here

- Developer documentation
- API reference
- Getting started guides
- Ported from 8gent-world docs

## Development

```bash
npm install
npm run dev
```

## Deploy

Deployed automatically via Vercel on push to main.
